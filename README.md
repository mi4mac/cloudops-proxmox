# CloudOPS Proxmox Pack

FortiSOAR content pack for Proxmox VE: request, approve, provision, and destroy VMs and LXC containers via playbooks.

⚠️ **PoC Use Only** - This package is intended for Proof of Concept use only. Not recommended for production due to security concerns (SSH password auth, no API, AD limitations). See [README.md](CloudOPS-Prx-pack-install/README.md) for details.

- **FortiSOAR**: 7.2.0+
- **Import**: Use the **CloudOPS-Prx-pack-install** folder (or its ZIP) in FortiSOAR Content Hub.
- **Scripts**: Deploy the **scripts/** folder to your Proxmox host (see `scripts/DEPLOY_ON_PROXMOX.txt`).

Full documentation: [CloudOPS-Prx-pack-install/README.md](CloudOPS-Prx-pack-install/README.md)
